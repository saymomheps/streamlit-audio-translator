# 🎵 অডিও ট্রান্সলেটর + টেক্স টু স্পিচ

এটি একটি Streamlit অ্যাপ্লিকেশন যা অডিও ফাইলকে টেক্সে রূপান্তরিত করে, অনুবাদ করে এবং টেক্সকে হাই কোয়ালিটি অডিওতে রূপান্তরিত করে।

## ✨ বৈশিষ্ট্য

⭐ **অডিও থেকে টেক্স:**
- OpenAI Whisper API দিয়ে টেক্সে রূপান্তর
- একাধিক ভাষা সমর্থন

🌐 **অনুবাদ:**
- GPT-3.5-turbo দিয়ে একাধিক ভাষায় অনুবাদ
- উচ্চ মানের অনুবাদ

🎙️ **টেক্স থেকে অডিও:**
- OpenAI TTS (Text-to-Speech) - হাই কোয়ালিটি
- একাধিক ভয়েস অপশন
- স্পীড নিয়ন্ত্রণ

## প্রয়োজনীয় জিনিস

- Python 3.8+
- OpenAI API Key

## ইনস্টলেশন

### স্টেপ ১: রিপোজিটরি ক্লোন করুন

```bash
git clone https://github.com/saymomheps/streamlit-audio-translator.git
cd streamlit-audio-translator
```

### স্টেপ २: Virtual Environment তৈরি করুন

```bash
python -m venv venv

# Windows এ:
venv\Scripts\activate

# macOS/Linux এ:
source venv/bin/activate
```

### স্টেপ ३: প্যাকেজ ইনস্টল করুন

```bash
pip install -r requirements.txt
```

### স্টেপ ४: অ্যাপ চালান

```bash
streamlit run app.py
```

ব্রাউজারে খুলবে: `http://localhost:8501`

## 🔑 OpenAI API Key পাওয়ার উপায়

### ১. OpenAI এ যান
- ব্রাউজারে যান: **https://platform.openai.com/api-keys**

### २. লগইন করুন
- আপনার OpenAI অ্যাকাউন্ট দিয়ে লগইন করুন
- নেই থাকলে ফ্রি অ্যাকাউন্ট তৈরি করুন: https://platform.openai.com/signup

### ३. নতুন API Key তৈরি করুন
- **"Create new secret key"** বাটনে ক্লিক করুন
- কী স্বয়ংক্রিয়ভাবে কপি হবে
- সেটি নিরাপদ জায়গায় সংরক্ষণ করুন

### ४. অ্যাপে ব্যবহার করুন
- অ্যাপের সাইডবারে API Key পেস্ট করুন

⚠️ **গুরুত্বপূর্ণ সুরক্ষা টিপস:**
- এই কী কখনো গিটহাব বা অন্য জায়গায় শেয়ার করবেন না
- পরিবেশ ভেরিয়েবলে সংরক্ষণ করুন (.env ফাইলে)

## 💳 খরচ

- **Whisper API:** প্রতি মিনিট $0.006 (অনুমানিত)

### উদাহরণ:

- অনুবাদ করুন: ~$0.001-0.005
- 1000 অক্ষরের অডিও তৈরি করুন: ~$0.015
- **মোট:** প্রতি অপারেশন ~$0.02-0.03

## ব্যবহারের নির্দেশনা

### অডিও থেকে টেক্স এবং অনুবাদ:

1. **"📤 অডিও থেকে টেক্স" ট্যাবে যান**
2. **অডিও ফাইল আপলোড করুন** (MP3, WAV, OGG বা M4A)
3. **"🔄 ট্রান্সক্রাইব করুন" ক্লিক করুন**
4. **"🌐 অনুবাদ করুন" ক্লিক করুন**
5. **ফলাফল দেখুন এবং ডাউনলোড করুন**

### টেক্স থেকে অডিও:

1. **"✍️ টেক্স থেকে অডিও" ট্যাবে যান**
2. **টেক্স এন্টার করুন**
3. **সাইডবারে ভয়েস এবং স্পীড নির্বাচন করুন**
4. **"🎙️ অডিও জেনারেট করুন" ক্লিক করুন**
5. **অডিও শুনুন এবং ডাউনলোড করুন**

## উপলব্ধ ভয়েস

- **alloy** - নিরপেক্ষ মানুষের ভয়েস
- **echo** - সামান্য নাটকীয় ভয়েস
- **fable** - গল্প বলার জন্য আদর্শ
- **onyx** - গভীর পুরুষ ভয়েস
- **nova** - প্রাকৃতিক মহিলা ভয়েস
- **shimmer** - উজ্জ্বল মহিলা ভয়েস
